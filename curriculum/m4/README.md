# M4 · 主題提煉 · Theme Crafting · テーマ抽出

## Phase
Builder (Phase 2, weeks 1–2).

## Time budget
- Content: 12 hours workshop.
- Validation: 3-reviewer consensus on one theme.

## Learning outcome
Trainee extracts a one-sentence theme from a real exhibit's primary-source material and defends it under reviewer challenge.

## Maps to capabilities
Layer 02 capability **02 Theme Crafting** (advanced; M1 introductory).

## Sample artifact (v0.1)
- Trainee selects an exhibit (from a list of 5 candidates the SME pre-curates).
- Trainee reads primary-source material (curator's notes, original cataloguing, ~30 pages).
- Trainee writes a theme sentence + defends in 5-minute workshop.
- 3 internal reviewers (each L3+) vote pass/fail.

**SME-pending**: the 5 candidate exhibits + their primary-source packets.

## Validation rubric
- **Present**: Pass by 3/3 reviewers; theme is propositional, defensible, distinct from a category label.
- **Partial**: Pass by 2/3.
- **Absent**: 0–1 of 3.

## Dependencies
M1 (TORE basics).

## SME content still needed

- ⛔ Reviewer rubric (consensus criteria, not just preference).
- ⛔ Past-trainee theme exemplars (3 strong, 3 weak — for calibration).
