# M5 · 跨文化敏感度 · Cross-cultural Sensitivity · 異文化への感受性

## Phase
Builder (Phase 2, weeks 3–6).

## Time budget
- Content: 24 hours classroom + simulations.
- Validation: scenario simulation, zero protocol violations.

## Learning outcome
Trainee identifies and respects cultural register expected by visitors from TW / JP / DE / CN / US-AZ; adjusts address, gesture, proxemics, and example choice in real time.

## Maps to capabilities
Layer 02 capability **08 Cross-cultural Sensitivity** (primary); **05 Audience Reading**, **07 Interactive Adaptation** (supporting).

## Sample artifact (v0.1)
- Per region: a one-page **taboo card** listing 5–8 culturally-sensitive cues to avoid.
- Per region: a **gesture map** listing 3–5 gestures with different meanings.
- Per region: a **proxemics chart** showing comfortable distance ranges.
- Simulation: trainee receives a mock VIP of named region; assessor scores protocol fidelity.

**SME-pending — THIS IS THE LARGEST SME ASK IN THE CURRICULUM.** James (#8) flagged that the deck presents this module as if it were content; the content does not exist yet.

## Validation rubric
- **Present**: Zero protocol violations across 3 simulations (one TW, one JP, one DE; CN and US-AZ added in V3).
- **Partial**: 1–2 minor violations; debrief + retry.
- **Absent**: ≥ 3 violations or one major (e.g. inappropriate touch, taboo gesture).
